# Quick Start: Structure Mode

Get started with the document structure feature in 5 minutes.

## Step 1: Generate an Outline

1. Go to http://localhost:8000/writer
2. Write or paste some content
3. Click the **AI** button
4. Select **"Outline"** from the mode dropdown
5. Click **"Generate"**

The outline preview page will open automatically showing your generated structure.

## Step 2: Review and Approve

1. Expand chapters and sections to review the structure
2. Check the statistics (chapters, sections, beats)
3. Click **"‚úì Approve & Convert to Document"**

You'll be taken to the Structure page with your outline converted to editable nodes.

## Step 3: Edit Your Document

1. Click any node in the left outline panel to select it
2. Edit the title in the top input field
3. Edit content in the textarea below
4. Create new nodes using toolbar buttons or keyboard shortcuts:
   - `Cmd/Ctrl+Alt+1` - New Chapter
   - `Cmd/Ctrl+Alt+2` - New Section
   - `Cmd/Ctrl+Alt+3` - New Subsection

## Step 4: Preview and Export

1. Click **"üìÑ Preview Document"** to see the full rendered document
2. Click **"‚úì Approve"** to send to writer home, or **"‚úï Close"** to continue editing
3. Use the **Export** dropdown to export as JSON, Markdown, Plain Text, or LaTeX

## Step 5: Navigate Between Modes

- **Structure ‚Üí Writer**: Click **"‚Üê Writer Home"** button
- **Writer ‚Üí Structure**: Click **"Structure"** button
- All your work is automatically saved and restored!

## Tips

- **All nodes are editable**: Click any node to edit its title and content
- **Auto-save**: Changes save every 1 second automatically
- **Persistence**: Your work persists across navigation, page reloads, and browser sessions
- **Drag to reorder**: Drag nodes in the outline panel to reorder them
- **Search**: Use the search box in the outline panel to find nodes quickly
